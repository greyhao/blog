一些命令

运行测试用例

`npx hardhat test`



启动本地网络

`npx hardhat node`



部署合约（本地网络需要运行着）

`npx hardhat run scripts/deploy.js --network localhost` //  --network 指定网络



启动测试服务

`npm run dev`



生成构建并运行

`npm run build && npm start`
